# 🩺 Doctor Patient Management System

A Java mini project that models the relationship between Doctors and Patients using Object-Oriented Programming concepts like encapsulation and bidirectional class interaction.

## 💡 Features
- Each doctor can treat up to 3 patients.
- Each patient can visit up to 3 doctors.
- Bidirectional relationship: Doctor ↔ Patient.
- Uses Scanner for real-time user input.

## 🛠️ Technologies Used
- Java
- OOP (Encapsulation, Bidirectional Association)
- Scanner (for interactive input)
